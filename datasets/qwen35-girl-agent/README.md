# Qwen 3.5 9B Instruct dataset для girl-agent

Датасет предназначен для дообучения небольшой instruct-модели под поведение `girl-agent`: Telegram-персона с состоянием, памятью, стадиями отношений, конфликтами, занятостью, сном и анти-AI стилем.

## Файлы

- `train.jsonl` — 72 записи для SFT.
- `eval.jsonl` — 8 записей для проверки после дообучения.
- `manifest.json` — состав датасета и метаданные генерации.

Формат каждой строки совместим с обычным chat-template обучением Qwen:

```json
{
  "messages": [
    { "role": "system", "content": "..." },
    { "role": "user", "content": "..." },
    { "role": "assistant", "content": "..." }
  ],
  "metadata": { "category": "..." }
}
```

Для обучения применяй chat template токенизатора Qwen, не склеивай роли вручную, если фреймворк уже умеет `messages`.

## Что покрыто

- короткие живые ответы без помощникового тона;
- длинноконтекстные диалоги с памятью и несколькими пузырями через `---`;
- генерация `persona.md`, `speech.md`, `communication.md`, `busySchedule`;
- внутренний `behavior-tick`: `reply`, `ignore`, `left-on-read`, реакции, задержки, `moodDelta`;
- дневные сводки, рефлексия, memory palace, запрет выдумывать факты;
- agenda и проактивные сообщения без сухого календарного поведения;
- исправление AI-slop: «конечно», «понимаю», markdown, лишняя вежливость, странный сленг.

## Стилевые ограничения

Цель — русскоязычная Telegram-переписка без англицизмов и без странного сленга. Эталонные ответы учат модель:

- не быть ассистентом и не обслуживать пользователя;
- не отвечать всегда, если по состоянию уместен игнор;
- не выдавать сердечки на холодных стадиях;
- не выдумывать память;
- писать коротко, с маленькой буквы, без markdown;
- использовать `---` только как разделитель пузырей сообщения.

## Генерация через ClaudeHub

Скрипт генерации лежит в `scripts/generate-qwen-dataset.mjs` и использует ClaudeHub как Anthropic-совместимый API:

```bash
export CLAUDEHUB_API_KEY="..."
node scripts/generate-qwen-dataset.mjs
```

Ключ нельзя хранить в репозитории, `.env` или датасете. Перед коммитом проверь, что в файлах нет ключа:

```bash
grep -R "sk-" datasets/qwen35-girl-agent scripts/generate-qwen-dataset.mjs
```

## Рекомендуемая проверка качества

После дообучения прогони `eval.jsonl` и вручную проверь:

1. JSON-ответы парсятся без пояснений вокруг.
2. Ответы девушки не содержат markdown, списков и помощниковых фраз.
3. Холодные стадии не дают тёплые реакции.
4. Конфликт и сон реально приводят к сухости, задержкам или игнору.
5. В long-term memory попадают только факты, явно сказанные пользователем.
