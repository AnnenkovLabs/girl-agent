# Exa Search

Отдельный `.gaa` аддон для тихого web-поиска через Exa.

## Что делает

- агент сам решает, когда нужен свежий внешний контекст;
- ищет новости, версии, цены, релизы, сайты, ссылки и факты, которые нельзя надёжно знать из памяти;
- использует найденное как внутренний контекст;
- не пишет пользователю «я поискала», «по данным Exa» или похожие фразы, если пользователь прямо не просит ссылку/источник.

## Как включить

1. Получи ключ Exa: https://dashboard.exa.ai/api-keys
2. В WebUI открой Addons и установи `exa-search.gaa`.
3. Открой настройки аддона, вставь ключ в поле `Exa API key` и нажми «Сохранить».

Если поле пустое, аддон попробует взять ключ из `EXA_API_KEY` окружения.

## Настройки

- `enabled` — включает/выключает тихий поиск.
- `apiKey` — Exa API key прямо в настройках WebUI.
- `maxResults` — сколько результатов читать за один поиск, от 1 до 5.
- `searchType` — `auto`, `fast` или `neural`.

По умолчанию используется `auto` и 3 результата.
